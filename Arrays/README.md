# Arrays

This directory contains Python implementations of common array-related algorithms and problems.

## Contents

### Anagram Check
Checks if two strings are anagrams of each other.
- [Anagram_Check_Sorted_Sol.py](Anagram_Check_Sorted_Sol.py): Solution using sorting.
- [Anagram_Check_manual_Sol.py](Anagram_Check_manual_Sol.py): Solution using a manual character count (dictionary).

### Find Missing Element
Find the missing element in a shuffled second array.
- [ArrayFindTheMissingElement_XOR_sol.py](ArrayFindTheMissingElement_XOR_sol.py): Solution using XOR bitwise operation.
- [ArrayFindTheMissingElement_brute_force_sol.py](ArrayFindTheMissingElement_brute_force_sol.py): Brute force solution.
- [ArrayFindTheMissingElement_hash_table_sol.py](ArrayFindTheMissingElement_hash_table_sol.py): Solution using a hash table (dictionary/collections.defaultdict).
- [ArrayFindTheMissingElement_takingSumandSubtract_sol.py](ArrayFindTheMissingElement_takingSumandSubtract_sol.py): Solution by comparing sums of both arrays.

### Array Pair Sum
Given an integer array, output all unique pairs that sum up to a specific value `k`.
- [ArrayPairSumSol.py](ArrayPairSumSol.py): Implementation using a set for tracking seen numbers.
