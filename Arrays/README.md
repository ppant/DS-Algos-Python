# Arrays

This directory contains Python implementations of common array-based algorithms and problems.

## Contents

- [Anagram Check (Sorted Solution)](Anagram_Check_Sorted_Sol.py): Checks if two strings are anagrams by comparing their sorted versions. Time complexity: $O(n \log n)$.
- [Anagram Check (Manual Solution)](Anagram_Check_manual_Sol.py): Checks if two strings are anagrams using a hash table (dictionary). Time complexity: $O(n)$.
- [Array Find Missing Element (XOR Solution)](ArrayFindTheMissingElement_XOR_sol.py): Efficiently finds a missing element using bitwise XOR. Time complexity: $O(n)$.
- [Array Find Missing Element (Brute Force Solution)](ArrayFindTheMissingElement_brute_force_sol.py): Finds a missing element by sorting and comparing arrays. Time complexity: $O(n \log n)$.
- [Array Find Missing Element (Hash Table Solution)](ArrayFindTheMissingElement_hash_table_sol.py): Finds a missing element using a hash table. Time complexity: $O(n)$.
- [Array Find Missing Element (Sum/Subtract Solution)](ArrayFindTheMissingElement_takingSumandSubtract_sol.py): Finds a missing element using the difference between array sums. Time complexity: $O(n)$.
- [Array Pair Sum Solution](ArrayPairSumSol.py): Finds unique pairs that sum to $k$ using a set. Time complexity: $O(n)$.
