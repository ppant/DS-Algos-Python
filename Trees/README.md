# Binary Trees and Searching

This directory contains Python implementations of binary trees, binary search trees (BST), and related algorithms.

## Contents

- [Binary Heap](BinaryHeapImple.py): Implementation of a Min-Heap.
- [Binary Search (Iterative)](BinarySearchImple.py): Efficiently searching a sorted list in $O(\log n)$ time using iteration.
- [Binary Search (Recursive)](BinarySearchRecursiveImple.py): Searching a sorted list in $O(\log n)$ time using recursion.
- [BST Check (Solution 1)](BinarySearchTreeCheckImpleSol1.py): Validating a BST using an in-order traversal approach.
- [BST Check (Solution 2)](BinarySearchTreeCheckImpleSol2.py): Validating a BST by keeping track of the min and max allowed values for each node.
- [Binary Search Tree Implementation](BinarySearchTreesImple.py): A comprehensive BST with search, insert, and delete operations. Average Time Complexity: $O(\log n)$, Worst: $O(n)$.
- [Tree Level Order Print](TreeLevelOrderPrintImple.py): Printing a tree level by level (Breadth-First Search).
- [Tree Representation (Nodes & References)](TreeRepresentationWithNodesReferences.py): Representing a tree using a `Node` class.
- [Trim a Binary Search Tree](TrimBinarySearchTreeImple.py): Trimming a BST so all its elements are within a range $[L, R]$.
- [Tree Representation (List of Lists)](BuildTreeTest.py): A simpler representation of a binary tree using nested lists.
