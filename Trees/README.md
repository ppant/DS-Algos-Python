# Trees

This directory contains Python implementations of tree-based data structures and algorithms.

## Contents

- [Binary Search Tree Implementation](BinarySearchTreesImple.py): Comprehensive BST implementation. $O(h)$, where $h$ is height.
- [BST Validation (Solution 1)](BinarySearchTreeCheckImpleSol1.py): Validates a BST using in-order traversal. $O(n)$.
- [BST Validation (Solution 2)](BinarySearchTreeCheckImpleSol2.py): Validates a BST using range constraints. $O(n)$.
- [Binary Search (Iterative)](BinarySearchImple.py): Iterative implementation of binary search. $O(\log n)$.
- [Binary Search (Recursive)](BinarySearchRecursiveImple.py): Recursive implementation of binary search. $O(\log n)$.
- [Binary Heap Implementation](BinaryHeapImple.py): Min-heap implementation. $O(\log n)$ for insert/extract.
- [Tree Level Order Print](TreeLevelOrderPrintImple.py): Prints a tree in level order (BFS). $O(n)$.
- [Trim BST](TrimBinarySearchTreeImple.py): Trims a BST to a given range $[min, max]$. $O(n)$.
- [Tree Representation (Nodes & References)](TreeRepresentationWithNodesReferences.py): Basic tree using node objects.
- [Tree Representation (List of Lists)](buildTreeTest.py): Simple tree representation using nested lists.
