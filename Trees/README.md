# Trees

This directory contains Python implementations of tree-based data structures and algorithms.

## Contents

- [Binary Heap Implementation](BinaryHeapImple.py): Implementation of a Min-Heap. Time Complexity: $O(\log n)$ for insert and extract-min.
- [Binary Search (Iterative)](BinarySearchImple.py): Iterative implementation of the Binary Search algorithm. Time Complexity: $O(\log n)$.
- [Binary Search (Recursive)](BinarySearchRecursiveImple.py): Recursive implementation of the Binary Search algorithm. Time Complexity: $O(\log n)$.
- [Binary Search Tree Check (Solution 1)](BinarySearchTreeCheckImpleSol1.py): Validates a BST using in-order traversal. Time Complexity: $O(n)$.
- [Binary Search Tree Check (Solution 2)](BinarySearchTreeCheckImpleSol2.py): Validates a BST by checking if each node falls within a valid range. Time Complexity: $O(n)$.
- [Binary Search Trees Implementation](BinarySearchTreesImple.py): Comprehensive implementation of a Binary Search Tree (BST). Time Complexity: $O(\log n)$ average, $O(n)$ worst case.
- [Build Tree Test](BuildTreeTest.py): Demonstration of building a tree using a list of lists approach.
- [Tree Level Order Print](TreeLevelOrderPrintImple.py): Prints a tree's nodes level by level (Breadth-First Search). Time Complexity: $O(n)$.
- [Tree Representation (Nodes and References)](TreeRepresentationWithNodesReferences.py): Basic tree structure using node objects and child references.
- [Trim a Binary Search Tree](TrimBinarySearchTreeImple.py): Removes nodes from a BST that fall outside a specified range $[L, R]$. Time Complexity: $O(n)$.
