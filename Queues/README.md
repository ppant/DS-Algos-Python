# Queues

This directory contains Python implementations of the Queue data structure.

## Contents

- [Queue Implementation](QueueImple.py): Basic implementation of a Queue (FIFO) using a Python list. $O(n)$ for dequeue (due to list shifting), $O(1)$ for enqueue.
- [Queue with Two Stacks](QueueWith2StacksImple.py): Implementation of a Queue using two Stacks. Amortized $O(1)$ for both enqueue and dequeue.
