# Queues

This directory contains Python implementations of the Queue data structure.

## Contents

- [Queue Implementation](QueueImple.py): Basic Queue implementation using a Python list (FIFO). Time Complexity: $O(1)$ for enqueue/dequeue (amortized).
- [Queue with Two Stacks](QueueWith2StacksImple.py): Implementing a Queue using two Stacks. Time Complexity: $O(1)$ amortized for enqueue/dequeue.
