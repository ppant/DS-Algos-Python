# Queues

This directory contains Python implementations of the Queue data structure.

## Contents

- [Queue Implementation](QueueImple.py): Basic implementation of a Queue using a Python list. Time Complexity: $O(n)$ for enqueue, $O(1)$ for dequeue.
- [Queue with Two Stacks](QueueWith2StacksImple.py): Implementing a FIFO queue using two stacks. Time Complexity: Amortized $O(1)$ for both enqueue and dequeue.
