# Queues

This directory contains Python implementations of the Queue data structure.

## Contents

- [Queue Implementation](QueueImple.py): Basic implementation of a FIFO queue using a Python list. $O(n)$ for enqueue, $O(1)$ for dequeue.
- [Queue with Two Stacks](QueueWith2StacksImple.py): Implements a queue using two stacks. $O(1)$ amortized for both enqueue and dequeue.
