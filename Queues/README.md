# Queues

This directory contains Python implementations of the Queue data structure.

## Contents

- [Queue Implementation](QueueImple.py): Implementation of a Queue using a Python list. Complexity: $O(n)$ for `enqueue`, $O(1)$ for `dequeue`.
- [Queue with Two Stacks](QueueWith2StacksImple.py): Implements a Queue using two stacks. Complexity: $O(1)$ amortized for both operations.
