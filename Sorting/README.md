# Sorting

This directory contains Python implementations of common sorting algorithms.

## Contents

- **Bubble Sort**: $O(n^2)$ worst and average case, $O(n)$ best case.
  - `BubbleSortImple.py`
- **Selection Sort**: $O(n^2)$ for all cases.
  - `SelectionSortImple.py`
- **Insertion Sort**: $O(n^2)$ worst and average case, $O(n)$ best case.
  - `InsertionSortImple.py`
- **Merge Sort**: $O(n \log n)$ for all cases.
  - `MergeSortImple.py`
- **Quick Sort**: $O(n^2)$ worst case, $O(n \log n)$ average/best case.
  - `QuickSortImple.py`
- **Shell Sort**: Between $O(n)$ and $O(n^2)$ depending on increment sequence.
  - `ShellSortImple.py`
