# Sorting Algorithms

This directory contains Python implementations of various sorting algorithms.

## Contents

- [Bubble Sort](BubbleSortImple.py): Implementation of Bubble Sort. $O(n^2)$
- [Selection Sort](SelectionSortImple.py): Implementation of Selection Sort. $O(n^2)$
- [Insertion Sort](InsertionSortImple.py): Implementation of Insertion Sort. $O(n^2)$
- [Shell Sort](ShellSortImple.py): Implementation of Shell Sort. $O(n^2)$
- [Merge Sort](MergeSortImple.py): A recursive "divide and conquer" algorithm. $O(n \log n)$
- [Quick Sort](QuickSortImple.py): Implementation of Quick Sort. $O(n \log n)$ average.
