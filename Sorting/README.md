# Sorting Algorithms

This directory contains Python implementations of various sorting algorithms with explanations of their complexities.

## Contents

- [Bubble Sort](BubbleSortImple.py): Implementation of Bubble Sort. Time Complexity: $O(n^2)$.
- [Selection Sort](SelectionSortImple.py): Implementation of Selection Sort. Time Complexity: $O(n^2)$.
- [Insertion Sort](InsertionSortImple.py): Implementation of Insertion Sort. Time Complexity: $O(n^2)$.
- [Shell Sort](ShellSortImple.py): Implementation of Shell Sort. Time Complexity: $O(n^2)$ worst-case.
- [Merge Sort](MergeSortImple.py): Implementation of Merge Sort. Time Complexity: $O(n \log n)$.
- [Quick Sort](QuickSortImple.py): Implementation of Quick Sort. Time Complexity: $O(n \log n)$ average.
