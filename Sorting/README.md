# Sorting Algorithms

This directory contains Python implementations of various sorting algorithms with complexity analysis.

## Contents

- [Bubble Sort](BubbleSortImple.py): Implementation of Bubble Sort. Complexity: $O(n^2)$
- [Selection Sort](SelectionSortImple.py): Implementation of Selection Sort. Complexity: $O(n^2)$
- [Insertion Sort](InsertionSortImple.py): Implementation of Insertion Sort. Complexity: $O(n^2)$
- [Shell Sort](ShellSortImple.py): Implementation of Shell Sort (diminishing increment sort). Complexity: $O(n^2)$ worst-case.
- [Merge Sort](MergeSortImple.py): A recursive "divide and conquer" algorithm. Complexity: $O(n \log n)$
- [Quick Sort](QuickSortImple.py): Implementation of Quick Sort. Complexity: $O(n \log n)$ average.
