# Sorting Algorithms

This directory contains several common sorting algorithm implementations in Python.

## Contents

### Bubble Sort
- **File**: `BubbleSortImple.py`
- **Complexity**: $O(n^2)$ worst and average case, $O(n)$ best case.

### Selection Sort
- **File**: `SelectionSortImple.py`
- **Complexity**: $O(n^2)$ for all cases.

### Insertion Sort
- **File**: `InsertionSortImple.py`
- **Complexity**: $O(n^2)$ worst and average case, $O(n)$ best case.

### Merge Sort
- **File**: `MergeSortImple.py`
- **Complexity**: $O(n \log n)$ for all cases.

### Quick Sort
- **File**: `QuickSortImple.py`
- **Complexity**: $O(n^2)$ worst case, $O(n \log n)$ average/best case.

### Shell Sort
- **File**: `ShellSortImple.py`
- **Complexity**: Between $O(n)$ and $O(n^2)$ depending on increment sequence.
