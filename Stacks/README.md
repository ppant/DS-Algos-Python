# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic implementation of a LIFO stack. Time Complexity: $O(1)$ for push/pop.
- [Balanced Parentheses Check](BalanceParenthesesCheckImple.py): Uses a stack to check if brackets are balanced. Time Complexity: $O(n)$.
