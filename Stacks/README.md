# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic Stack with `push`, `pop`, and `peek` operations. Complexity: $O(1)$
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Uses a stack to check if brackets are balanced in a string. Complexity: $O(n)$
