# Stacks

This directory contains Python implementations of common stack-based data structures.

## Contents

- **Stack Implementation**: Basic stack operations (LIFO): `push`, `pop`, `peek`, `isEmpty`, `size`.
  - `StackImple.py`
- **Balanced Parentheses Check**: Check if a string of opening and closing parentheses is balanced.
  - `BalanceParenthlessCheckImple.py`
