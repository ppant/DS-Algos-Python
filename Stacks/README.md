# Stacks

This directory contains Python implementations of the Stack data structure and related problems.

## Contents

- [Stack Implementation](StackImple.py): Basic LIFO stack implementation. Time Complexity: $O(1)$ for push and pop.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Using a stack to check if a string of parentheses is balanced. Time Complexity: $O(n)$.
