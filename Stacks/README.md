# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic implementation of a LIFO (Last-In-First-Out) stack using a Python list. Includes `push` ($O(1)$), `pop` ($O(1)$), `peek` ($O(1)$), `isEmpty` ($O(1)$), and `size` ($O(1)$).
- [Balanced Parentheses Check](BalanceParenthesesCheckImple.py): Uses a stack to check if a string of opening and closing parentheses (round, square, and curly) is balanced. $O(n)$
