# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic LIFO stack using a Python list. Time complexity: $O(1)$ for all operations.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Uses a stack to check for balanced parentheses. Time complexity: $O(n)$.
