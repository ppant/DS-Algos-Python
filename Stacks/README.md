# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic implementation of a Stack (LIFO) using a Python list. $O(1)$ for push/pop.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Uses a stack to check if a string of parentheses is balanced. $O(n)$.
