# Stacks

This directory contains Python implementations of the Stack data structure.

## Contents

- [Stack Implementation](StackImple.py): Basic LIFO (Last-In-First-Out) stack implementation. Time Complexity: $O(1)$ for all operations.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Uses a stack to check if a string of parentheses is balanced. Time Complexity: $O(n)$.
