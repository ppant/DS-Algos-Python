# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic implementation of a LIFO (Last-In-First-Out) stack using a Python list. Time Complexity: $O(1)$ for `push`, `pop`, `peek`.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Uses a stack to check if a string of opening and closing parentheses is balanced. Time Complexity: $O(n)$.
