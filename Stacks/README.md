# Stacks

This directory contains Python implementations of the Stack data structure and its applications.

## Contents

- [Stack Implementation](StackImple.py): Basic implementation of a LIFO stack using a Python list. $O(1)$ for push and pop.
- [Balanced Parentheses Check](BalanceParenthlessCheckImple.py): Checks if parentheses are balanced using a stack. $O(n)$
