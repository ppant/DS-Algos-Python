# Linked Lists

This directory contains Python implementations of various types of linked lists and related algorithms.

## Contents

- [Singly Linked List Implementation](SingleLinkedListImple.py): Basic implementation of a singly linked list node and basic linkage. $O(1)$ for insertion at head.
- [Doubly Linked List Implementation](DoublyLinkedListImple.py): Basic implementation of a doubly linked list node with `prev` and `next` pointers. $O(1)$ for insertion at head/tail.
- [Singly Linked List Cycle Check](SinglyLinkedListCycleCheckImple.py): Implements Floyd's Cycle-Finding Algorithm (two pointers) to detect cycles in a linked list. $O(n)$
- [Linked List Reversal](LinkedListReversal.py): Reverses a singly linked list in-place. $O(n)$
- [Nth to Last Node](LinkedListNthToLastNode.py): Finds the $n$-th to last node in a singly linked list using two pointers. $O(n)$
