# Linked Lists

This directory contains Python implementations of various types of linked lists and related algorithms.

## Contents

- [Singly Linked List Implementation](SingleLinkedListImple.py): Basic implementation of a singly linked list node and basic linkage. Time Complexity: $O(1)$.
- [Doubly Linked List Implementation](DoublyLinkedListImple.py): Basic implementation of a doubly linked list node with `prev` and `next` pointers. Time Complexity: $O(1)$.
- [Singly Linked List Cycle Check](SinglyLinkedListCycleCheckImple.py): Implements Floyd's Cycle-Finding Algorithm (two pointers) to detect cycles in a linked list. Time Complexity: $O(n)$.
- [Linked List Reversal](LinkedListReversal.py): Reverses a singly linked list in-place. Time Complexity: $O(n)$.
- [Nth to Last Node](LinkedListNthToLastNode.py): Finds the $n$-th to last node in a singly linked list using two pointers. Time Complexity: $O(n)$.
