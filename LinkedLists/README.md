# Linked Lists

This directory contains Python implementations of various types of linked lists and related algorithms.

## Contents

- [Singly Linked List Implementation](SingleLinkedListImple.py): Basic implementation of a singly linked list. $O(1)$ for node operations.
- [Doubly Linked List Implementation](DoublyLinkedListImple.py): Basic implementation of a doubly linked list. $O(1)$ for node operations.
- [Singly Linked List Cycle Check](SinglyLinkedListCycleCheckImple.py): Implements Floyd's Cycle-Finding Algorithm to detect cycles. $O(n)$
- [Linked List Reversal](LinkedListReversal.py): Reverses a singly linked list in-place. $O(n)$
- [Nth to Last Node](LinkedListNthToLastNode.py): Finds the $n$-th to last node using two pointers. $O(n)$
