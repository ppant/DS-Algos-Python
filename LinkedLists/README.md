# Linked Lists

This directory contains Python implementations of Singly and Doubly Linked Lists and related problems.

## Contents

- [Singly Linked List Implementation](SingleLinkedListImple.py): Basic Singly Linked List with insertion and deletion. Time Complexity: $O(1)$ at head.
- [Doubly Linked List Implementation](DoublyLinkedListImple.py): Basic Doubly Linked List with forward and backward traversal. Time Complexity: $O(1)$ at head/tail.
- [Cycle Detection](SinglyLinkedListCycleCheckImple.py): Detect cycles using Floyd's Cycle-Finding Algorithm. Time Complexity: $O(n)$.
- [Reverse Linked List](LinkedListReversal.py): In-place reversal of a Singly Linked List. Time Complexity: $O(n)$.
- [Nth to Last Node](LinkedListNthToLastNode.py): Find the $n$-th node from the end of a Singly Linked List. Time Complexity: $O(n)$.
