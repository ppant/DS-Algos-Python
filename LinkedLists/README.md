# Linked Lists

This directory contains Python implementations of various types of linked lists and related algorithms.

## Contents

- [Singly Linked List Implementation](SingleLinkedListImple.py): Basic implementation of a singly linked list node. Complexity: $O(1)$ for node creation and linkage.
- [Doubly Linked List Implementation](DoublyLinkedListImple.py): Basic implementation of a doubly linked list node with `prev` and `next` pointers. Complexity: $O(1)$.
- [Singly Linked List Cycle Check](SinglyLinkedListCycleCheckImple.py): Implements Floyd's Cycle-Finding Algorithm (two pointers) to detect cycles. Complexity: $O(n)$ time, $O(1)$ space.
- [Linked List Reversal](LinkedListReversal.py): Reverses a singly linked list in-place. Complexity: $O(n)$ time, $O(1)$ space.
- [Nth to Last Node](LinkedListNthToLastNode.py): Finds the $n$-th to last node using two pointers. Complexity: $O(n)$ time, $O(1)$ space.
