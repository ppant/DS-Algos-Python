# Recursion

This directory contains Python implementations of problems solved using recursion and dynamic programming.

## Contents

- [Fibonacci Sequence (Iterative)](FibonacciSeqIterative.py): Iterative solution for Fibonacci numbers. Time Complexity: $O(n)$.
- [Fibonacci Sequence (Recursive)](FibonacciSeqRecursion.py): Recursive solution for Fibonacci numbers. Time Complexity: $O(2^n)$.
- [Fibonacci Sequence (Dynamic Programming)](FibonacciSeqDynamic.py): DP solution for Fibonacci numbers. Time Complexity: $O(n)$.
- [Coin Change Problem (Recursive)](CoinChangeProblemRecursion.py): Recursive solution for the coin change problem.
- [Coin Change Problem (Dynamic Programming)](CoinChangeProblemDynamic.py): DP solution for the coin change problem. Time Complexity: $O(n \cdot m)$.
- [Recursion Reverse String](RecursionReverseStr.py): Reversing a string using recursion. Time Complexity: $O(n)$.
- [Recursion String Permutation](RecursionStrPermutation.py): Finding all permutations of a string. Time Complexity: $O(n!)$.
- [Recursion Cumulative Sum](RecursionCumulativeSum.py): Calculating cumulative sum using recursion. Time Complexity: $O(n)$.
- [Recursion Sum of Digits](RecursionSumOfDigits.py): Calculating the sum of digits using recursion. Time Complexity: $O(\log n)$.
- [Recursion Word Split](RecursionWordSplit.py): DP solution for splitting words. Time Complexity: $O(n^2)$.
