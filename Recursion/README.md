# Recursion

This directory contains Python implementations of recursive algorithms and dynamic programming solutions.

## Contents

- [Fibonacci Sequence (Dynamic)](FibonacciSeqDynamic.py): Fibonacci sequence using memoization. $O(n)$
- [Fibonacci Sequence (Iterative)](FibonacciSeqIterative.py): Fibonacci sequence using an iterative approach. $O(n)$
- [Fibonacci Sequence (Recursion)](FibonacciSeqRecursion.py): Fibonacci sequence using simple recursion. $O(2^n)$
- [Coin Change (Dynamic)](CoinChangeProblemDynamic.py): Coin change problem using dynamic programming. $O(n \cdot m)$
- [Coin Change (Recursion)](CoinChangeProblemRecursion.py): Coin change problem using simple recursion.
- [Recursion Cumulative Sum](RecursionCumulativeSum.py): Calculates the cumulative sum using recursion. $O(n)$
- [Recursion Reverse String](RecursionReverseStr.py): Reverses a string using recursion. $O(n)$
- [Recursion String Permutation](RecursionStrPermutation.py): Generates all permutations of a string using recursion. $O(n!)$
- [Recursion Sum of Digits](RecursionSumOfDigits.py): Calculates the sum of digits using recursion. $O(\log_{10} n)$
- [Recursion Word Split](RecursionWordSplit.py): Splits a string into words based on a dictionary using recursion.
