# Recursion

This directory contains Python implementations of problems solved using recursion and dynamic programming.

## Contents

- [Fibonacci (Iterative)](FibonacciSeqIterative.py): Iterative implementation of Fibonacci. $O(n)$
- [Fibonacci (Recursive)](FibonacciSeqRecursion.py): Simple recursive implementation of Fibonacci. $O(2^n)$
- [Fibonacci (Dynamic Programming)](FibonacciSeqDynamic.py): Optimized Fibonacci using memoization. $O(n)$
- [Coin Change (Recursive)](CoinChangeProblemRecursion.py): Basic recursive solution for coin change. Exponential complexity.
- [Coin Change (Dynamic Programming)](CoinChangeProblemDynamic.py): Optimized solution for coin change using DP. $O(n \cdot m)$
- [Cumulative Sum](RecursionCumulativeSum.py): Computes the cumulative sum recursively. $O(n)$
- [Reverse a String](RecursionReverseStr.py): Reverses a string using recursive calls. $O(n)$
- [String Permutations](RecursionStrPermutation.py): Generates all permutations of a string. $O(n!)$
- [Sum of Digits](RecursionSumOfDigits.py): Calculates the sum of digits recursively. $O(\log_{10} n)$
- [Word Split](RecursionWordSplit.py): Determines if a string can be split into words. $O(n^2)$
