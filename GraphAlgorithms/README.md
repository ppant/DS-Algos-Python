# Graph Algorithms

This directory contains Python implementations of graph data structures and algorithms.

## Contents

- [Adjacency List Implementation](AdjacencyListGraphImple.py): Basic Graph ADT using an adjacency list.
- [Breadth First Search (BFS)](BFS.py): Implementation of BFS for solving the Word Ladder problem. $O(V+E)$.
- [Depth First Search (DFS General)](DFSGeneral.py): General implementation of DFS. $O(V+E)$.
- [The Knight's Tour Problem](TheKnightsTourProblem.py): Building the graph for the Knight's Tour problem.
- [DFS Knight's Tour](DFSImpleTheKnightsTourProblem.py): DFS-based solution for the Knight's Tour problem.
- [Word Ladder Problem](WordLadderProblem.py): Implementation of the Word Ladder graph construction.
