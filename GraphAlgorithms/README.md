# Graph Algorithms

This directory contains Python implementations of common graph-based algorithms and data structures.

## Contents

- [Adjacency List Implementation](AdjacencyListGraphImple.py): Implements the Graph Abstract Data Type (ADT) using an adjacency list.
- [Breadth First Search (BFS)](BFS.py): Implements BFS to solve the Word Ladder problem. Time Complexity: $O(V+E)$.
- [General Depth First Search (DFS)](DFSGeneral.py): Provides a general implementation of DFS. Time Complexity: $O(V+E)$.
- [DFS - Knight's Tour Problem](DFSImpleTheKnightsTourProblem.py): Implementation of DFS tailored to the Knight's Tour puzzle. Time Complexity: $O(k^N)$.
- [The Knight's Tour Problem](TheKnightsTourProblem.py): Generates the knight's move graph and solves the tour using DFS and backtracking.
- [Word Ladder Problem](WordLadderProblem.py): Focuses on building the word ladder graph. Time Complexity: $O(V+E)$.
