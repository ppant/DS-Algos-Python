# Graph Algorithms

This directory contains Python implementations of common graph data structures and algorithms.

## Contents

- **Adjacency List Implementation**: Basic structure for an adjacency list-based graph.
  - `AdjacencyListGraphImple.py`
- **Breadth First Search (BFS)**: Implementation of the BFS algorithm and its application to the Word Ladder Problem.
  - `BFS.py`
  - `WordLadderProblem.py`
- **Depth First Search (DFS)**: Implementation of the DFS algorithm and its application to the Knight's Tour Problem.
  - `DFSGeneral.py`
  - `DFSImpleTheKnightsTourProblem.py`
  - `TheKnightsTourProblem.py`
