# Deque

This directory contains a Python implementation of a Double-Ended Queue (Deque).

## Contents

- [Deque Implementation](DequeImple.py): A class-based implementation of a deque using a Python list.
  - `addFront()`: $O(1)$
  - `addRear()`: $O(n)$
  - `removeFront()`: $O(1)$
  - `removeRear()`: $O(n)$
  - `isEmpty()`: $O(1)$
  - `size()`: $O(1)$
