# Deque

This directory contains a Python implementation of the Deque (Double-Ended Queue) data structure.

## Contents

- [Deque Implementation](DequeImple.py): Implementation of a Deque using a Python list. Includes `addFront`, `addRear`, `removeFront`, `removeRear`, `isEmpty`, and `size` methods. Complexity: `addFront` and `removeFront` are $O(1)$, while `addRear` and `removeRear` are $O(n)$ due to list shifting.
