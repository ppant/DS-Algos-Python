# Deque

This directory contains Python implementations of the Deque (Double-Ended Queue) data structure.

## Contents

- [Deque Implementation](DequeImple.py): Basic implementation of a Deque using a Python list. Time Complexity: `addFront`/`removeFront` are $O(1)$, `addRear`/`removeRear` are $O(n)$ due to list shifting.
