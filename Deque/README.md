# Deque

This directory contains Python implementations of the Deque (Double-Ended Queue) data structure.

## Contents

- [Deque Implementation](DequeImple.py): Basic implementation of a Deque using a Python list. Complexity: $O(1)$ for `addFront`/`removeFront`, $O(n)$ for `addRear`/`removeRear` due to list shifting.
