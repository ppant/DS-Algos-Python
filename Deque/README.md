# Deque

This directory contains Python implementations of the Deque (Double-Ended Queue) data structure.

## Contents

- [Deque Implementation](DequeImple.py): Basic implementation of a Deque using a Python list.
  - `addFront`: $O(1)$
  - `addRear`: $O(n)$
  - `removeFront`: $O(1)$
  - `removeRear`: $O(n)$
  - `isEmpty`: $O(1)$
  - `size`: $O(1)$
